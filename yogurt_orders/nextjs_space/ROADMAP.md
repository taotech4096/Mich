# Product Roadmap

## Current Stage
- Website storefront and admin dashboard are running.
- Customers can register/login, place orders, and create subscriptions.
- Core operations exist (orders, customers, products, settings).

## Phase 1: Commerce Engine (Now)
### Goal
Secure payment + reliable recurring-order operations.

### Scope
1. Mercado Pago checkout integration.
2. Payment webhooks to automatically update order status.
3. Subscription auto-order generation job.
4. Admin visibility for payment and subscription execution status.
5. Baseline operational docs and environment setup.

### Success Criteria
- Customer can pay online and be redirected successfully.
- Order status updates from webhook without manual intervention.
- Active subscriptions create due orders via scheduled job.
- Team can operate the flow with a repeatable checklist.

## Phase 2: Marketing Assistant Platform
### Goal
Reduce manual social media work while keeping brand quality high.

### Scope
1. Content calendar and workflow states.
2. AI caption generation (Spanish-first, optional English variants).
3. Copy/paste export for assistants.
4. Optional direct posting integrations with approval workflow.
5. Basic performance dashboard by platform/content type.

### Success Criteria
- Assistant can produce and schedule weekly content plan in one place.
- Posting workflow is auditable (draft -> approved -> posted).
- Time spent per post is reduced consistently.

## Phase 3: Optimization & Scale
### Goal
Increase retention and operational efficiency.

### Scope
1. Delivery route optimization and delivery assignment tools.
2. Cohort metrics (retention, reorder rate, churn).
3. Subscription lifecycle automation (dunning, pause recovery).
4. Loyalty/referral campaigns.

### Success Criteria
- Lower churn and more active subscriptions.
- Faster fulfillment with fewer manual follow-ups.
- Clear KPI dashboard for weekly decision-making.
